# Leviathan Positioning

## One Sentence

Leviathan is a Solana-native constitutional execution and pre-trade decision infrastructure stack for autonomous agents and human operators.

## What It Is Not

Leviathan is not:

- a dashboard-only analytics product
- a generic token safety widget
- a wallet permission toy
- an always-on trading bot
- a routing or gas optimization layer

## What It Is

Leviathan sits one layer earlier and one layer deeper:

- earlier than execution, because it decides whether the action should exist at all
- deeper than dashboards, because it outputs machine-usable decisions and policy boundaries

## Core Product Thesis

As more capital is screened, allocated, and moved by software, the critical problem is no longer only discovering opportunities. The critical problem is deciding whether an action is legitimate, bounded, and accountable before it touches capital.

That is the layer Leviathan is built for.
